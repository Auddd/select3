### 实现功能
select3.js实现以下功能，且调用方便快捷:

- 完善select2不支持tree的缺陷
- 理论上支持N层级的树状下拉框
- 本地搜索
- 支持数据回显选中
- 支持设置单选多选
- 主JS压缩后只有8kb
